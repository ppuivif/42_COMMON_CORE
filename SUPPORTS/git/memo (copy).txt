-----------------------------------------------------------------------------------------------

Pour connaître les différentes branches dans un projet :

git branch

donne :
* main
PAP

NB : l'étoile (* main) indique la branche sur laquelle l'utilisateur est situé.

-----------------------------------------------------------------------------------------------

Pour créer une branche en local :

git branch Driss

-----------------------------------------------------------------------------------------------

Pour basculer sur une branche existant :

git checkout Driss

ou

git switch Driss

ensuite

git branch

donne :
main
PAP
*Driss

-----------------------------------------------------------------------------------------------

Pour enregistrer et créer une nouvelle version du projet en local :

git commit -m "nouvelle branche"

-----------------------------------------------------------------------------------------------

Pour envoyer les modifications sur le dépôt distant :


git push

-----------------------------------------------------------------------------------------------

Pour fusionner une branche avec la branche principale (une fois les modifications
opérationnelles sur la branche annexe) :

Se placer dans la branche qui va accueillir les évolutions :

git checkout main

Fusionner la branche annexe avec la branche principale :

git merge PAP

-----------------------------------------------------------------------------------------------

Pour quitter un merge en cours (en cas de conflit par exemple) :

git merge --quit

ou

git merge --abort


-----------------------------------------------------------------------------------------------
