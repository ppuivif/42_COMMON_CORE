  Description du projet :

      Créer une interface web faisant office de portfolio personnel.
      Cette interface devra permettre d'afficher, voire de lancer quelques projets réalisés au cours du tronc commun à 42.

    Compétences apportées par le projet :

      Réaliser un projet personnel hors cursus 42.
      La création de l'interface pourra permettre de se familiariser avec quelques technologies web.

    Technologies retenues :

      - HTML, CSS, JAVA.
      - ANGULAR (framework JAVASCRIPT écrit en TYPESCRIPT) pour le frontend.
      - TYPESCRIPT pour le backend.

    Technologies alternatives :

      - JAVA
      - SPRINGBOOT (Framework JAVA) pour le backend.
      - DJANGO (Framework PYTHON) pour le backend.
      - REACT (librairie JAVASCRIPT) pour le frontend.
      - NODE.JS (environnement d'exécution JAVASCRIPT) pour le backend.
