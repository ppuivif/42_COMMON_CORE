Mémo portant sur la création d'une page HTML

Créer un fichier .html :

Pour créer un fichier html dans VSCODE, il suffit de créer un nouveau fichier portant l'extension ".html".

Créer une structure HTML basique :

Suivre la procédure suivante :

Déclaration du type de document à savoir HTML5 (Document Type Declaration) :
      <!DOCTYPE html>
Cette première ligne permet au navigateur d'afficher correctement la page en lui indiquant qu'il s'agit d'un document HTML5.
En l'absence de cette ligne, le navigateur est susceptible de prendre en charge le document dans une version plus ancienne de HTML.
NB : cette déclaration n'est pas un tag.

Insertion d'un tag d'ouverture du document HTML, avec la spécification de la langue utilisée dans le contenu du document :
      <html lang="en">
lang="en" spécifie la langue utilisée (e.g., en pour English, fr pour French).
La spécification de la langue principale ne peut être défini que dans cette ligne.
En son absence, les navigateurs pourront chercher à déterminer la langue en fonction des paramètres de l'utilisateur ou du contenu, mais cela n'est pas toujours fiable.
Par ailleurs les technologies d'assistance peuvent utiliser par défaut une langue erronée ou non souhaitée.
Enfin la page risque de ne pas bénéficier des avantages du référencement (SEO).
En revanche, il est possible de définir des langues additionnelles de façon ponctuelle dans le fichier.
NB : un tag de fermeture marquera la fin du document (</html>).

Insertion d'un elément <head> :
Cet élément est un container pour les métadonnées, à savoir des données concernant le document mais qui ne sont pas affichées.
Cet élément commence par le tag :
      <head>
et se termine par le tag :
      </head>
Il contient une ligne permettant de définir l'encodage des caractères :
      <meta charset="UTF-8">
L'encodage retenu est conforme à l'UTF-8, qui supporte presque tous les caractères et symboles, ce qui garantit un affichage correct du texte.
L'élément <head> contient également une ligne permettant d'ajuster l'affichage de la page :
      <meta name="viewport" content="width=device-width, initial-scale=1.0">
L'affichage de la page s'adapte à la taille de l'écran (tel que celui des smartphones).
Sans cette ligne, la page risque ne pas s'afficher correctement sur les petits écrans.
L'élément <head> contient également la définition du titre de la page :
      <title>My First HTML Page</title></p>
Ce titre n'est pas en tête du document mais celui qui s'affiche dans l'onglet du navigateur ou comme titre dans les résultats des moteurs de recherche.

Insertion d'un élément <body> :
Cet élément contient ce qui sera affiché dans la page (texte, images, vidéos, liens).
Cet élément commence par le tag :
      <body>
et se termine par le tag :
      </body>

En conclusion, le bloc de code HTML à créer est le suivant :
            <!DOCTYPE html>
      <html lang="en">
      <head>
            <meta charset="UTF-8">
            <meta name="viewport" content="width=device-width, initial-scale=1.0">
            <title>Your Page Title</title>
      </head>
      <body>
                  Contenu de la page.
            </body>
      </html>
